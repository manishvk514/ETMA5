# Everything Media — final website package

This package contains the rebuilt static website and the handoff documentation for the premium visual direction.

## Files

- `index.html` — final website structure
- `style.css` — complete visual system and layout
- `script.js` — motion, cursor, featured-work switching, contact flow
- `vercel.json` — static hosting headers for Vercel
- `VISUAL_SYSTEM.md` — design language and section composition
- `SCROLL_CHOREOGRAPHY.md` — how the scroll is staged and timed
- `ASSET_SPECS.md` — what hero videos/images and project thumbnails to prepare
- `IMPLEMENTATION_NOTES.md` — where to replace assets, copy, links, and how to extend with real 3D

## What changed

This version keeps the original copy intact and changes the site architecture so it feels directed rather than template-like:

- calibration-style opening
- cinematic hero with framing guides
- thesis panel with restrained reveal
- archive-style work index
- sticky featured project stage
- horizontal reel corridor
- process cards that behave like storyboard boards
- quieter conviction section
- cleaner intake-style contact layout

## How to preview

Open `index.html` in the browser.

## How to deploy

Upload all files to a GitHub repo and import the repo into Vercel.
