# Visual system

## Overall direction

The site should feel like a directed film object, not a portfolio template. The guiding references are:

- a film contact sheet
- a director's treatment
- a luxury editorial object
- a screening-room interface

## Principles

1. Strong negative space
2. Few but intentional colors
3. Typography leads hierarchy
4. Motion reveals structure, not decoration
5. 3D feeling comes from depth and glass, not random objects

## Section visual logic

### Hero
- full-bleed image or slow-moving background plate
- frame guides and safe-area lines
- optical glass overlay
- large wordmark with controlled crop reveal
- restrained technical metadata

### Positioning
- almost empty stage
- thesis line-by-line through slit reveal
- supporting copy only after the statement lands

### Work index
- archive layout instead of a uniform grid
- different card proportions so the selection feels curated
- titles and metadata treated like film notes

### Featured stage
- selected project shown as a large framed plane
- slight depth and reflective lighting
- image, title, and metadata feel separated by layers

### Reel corridor
- vertical scroll moves a horizontal strip
- should feel like moving past mounted stills

### Method
- presented as craft boards, not a numbered list
- each step gets its own minimal visual motif

### Conviction
- large pause in the site rhythm
- almost black stage
- thesis presented like a title sequence

### Contact
- an intake room, not a generic form
- sparse, architectural field layout

## Typography

- display: elegant serif with high contrast and strong editorial character
- support: premium sans for technical UI and labels
- uppercase labels only where system-like behavior helps

## Color

- deep black base
- warm bone text
- one rust-red accent
- no broad colorful gradients beyond controlled lighting
